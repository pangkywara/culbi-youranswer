## рҹ“– What is Culbi?

**Culbi (Cultural Bridge)** is a mobile-first platform that transforms how people discover, learn about, and connect through the cultural heritage of **ASEAN nations**. Instead of being a traditional travel guide, Culbi turns cultural exploration into a gamified, social, and AI-powered experience.

Point your camera at a landmark вҖ” Culbi identifies it instantly using **Google Gemini AI**, enriches it with cultural stories, and rewards you with XP and collectible badges. Plan trips with an **AI concierge**, join location-based community groups right on the map, and compete on leaderboards вҖ” all while discovering the rich tapestry of Southeast Asian culture.

### рҹҺҜ The Problem We Solve

> Cultural heritage across ASEAN is incredibly rich but **fragmented and hard to access**. Travelers visit landmarks without understanding their cultural significance. Locals lack a platform to share and preserve their heritage. There's no engaging way to bridge this gap.

**Culbi bridges that gap** by making cultural discovery interactive, social, and rewarding.

## рҹҡҖ Getting Started

### Prerequisites

| Requirement | Version |
|------------|---------|
| Node.js | вүҘ 18.x |
| npm | вүҘ 9.x |
| Xcode | вүҘ 15 (iOS) |

| Google Maps API Key | [Get one here](https://console.cloud.google.com/) |

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/pangkywara/culbi-youranswer.git
cd culbi-youranswer

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env
# Edit .env with your keys (see Environment Variables below)

# 4. Generate native projects
npx expo prebuild

# 5. Install iOS pods
cd ios && pod install && cd ..
```

### Running the App

```bash
# iOS Simulator
npx expo run:ios

# Android Emulator
npx expo run:android

# Development server (Expo Go вҖ” limited features)
npm start
```

> **Note:** Google Maps and Landmark Detection require a **native dev build** (`npx expo run:ios/android`), not Expo Go.
